Easy to add twitter recent tweets on your WordPress site by using the Recent Tweets Widget plugin.

== Description ==

Twitter is one of the most powerful and highly influential social networking tool on the planet. 

<strong>: To find your oauth access and consumer tokens :</strong>

<ul>
	<li>You will need a Twitter account. If you don’t already have one, <a href="https://twitter.com/signup" target="_blank">create an account</a>.</li>
	<li>Go to <a href="https://dev.twitter.com/" target="_blank">https://dev.twitter.com</a> and <a href="https://dev.twitter.com/user/login" target="_blank">log-in</a>.</li>
	<li>Click “<a href="https://dev.twitter.com/apps">My Applications</a>” from the top right menu under your profile picture.</li>
	<li>Click the blue “Create a new application” button.</li>
	<li>Fill out the required information and create your application.</li>
	<li>When finished, click “Create my access token.”</li>
</ul>


= Features =
* Easy Twitter Tweets Widget Plugin is very easy to setup and use.

== Installation ==

1. Upload the `wp-recent-tweets` folder to the `/wp-content/plugins/` directory
2. Activate the WP Recent Tweets Widget plugin through the 'Plugins' menu in WordPress
3. Use the Recent Tweets Widget by going to the `Appearance > Widgets`
